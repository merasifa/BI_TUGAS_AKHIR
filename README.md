## Deskripsi Repository
Repository ini berisi hasil pengerjaan **Proyek Tugas Akhir** untuk **memenuhi Ujian Akhir Semester (UAS) mata kuliah Business Intelligence**.  
Proyek ini membahas pembangunan **Data Warehouse E-commerce European Fashion** menggunakan proses **Extract, Transform, Load (ETL)** dengan Pentaho Data Integration.

---

## Struktur Repository
Struktur folder pada repository ini adalah sebagai berikut:

- **pentaho/**  
  Berisi file ETL Pentaho dengan ekstensi `.ktr`.

- **db_ecommerce/**  
  Berisi database sumber (data asli) yang digunakan sebelum proses ETL.

- **dw_ecommerce/**  
  Berisi database hasil pengolahan yang telah dibentuk menjadi Data Warehouse.

- **laporan/**  
  Berisi dokumen laporan proyek Business Intelligence.
